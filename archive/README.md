# Archive

This directory keeps historical materials that are useful for traceability but
are not part of the current FM-AVSR delivery path.

## Current Entry Points

Use these files for the active branch:

- `../README.md`
- `../doc/fm_avsr_final_status_2026-06-04.md`
- `../doc/fm_avsr_audio_generation_architecture.md`
- `../doc/raw_video_avsr_recon_pipeline_usage.md`
- `../report/`
- `../poster/`

## Contents

- `docs_legacy/`: early StreamLip design notes, literature surveys, handover
  notes, and older experiment logs.
- `proposal_legacy/`: old proposal template and source files. These no longer
  describe the implemented system.
- `papers_legacy/`: local PDF references and paper collection used during
  exploration.

The archived files are intentionally left intact. Move a file back only if it is
updated to match the current recon-based FM-AVSR architecture.
